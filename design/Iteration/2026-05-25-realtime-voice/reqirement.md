# 需求

- 为学习中文的学生，开发和AI实时对话的功能，主要支持中文对话

# 目标

- 为国外的汉语学习者，在practice 页面，根据场景和不同的角色，提供和AI实时聊天的功能
- 对话完后可以导出，总结报告

# 功能

- 从首页跳转后，能带入参数区分不同的场景，根据不同的场景角色扮演，实时对话，例如首页中的：酒店入住，面试自我介绍，日常对话-点咖啡。
  - 如果有参数，则根据参数代表的主题场景，进行实时对话
  - 如果没有参数，则为自由聊天
- 参考功能豆包的实时聊天，能实时显示聊天记录
- 对话完后，可以根据聊天记录导出给出中文分析报告，分析中文相关的语法、词语、声调等问题。
- 该次对话保存到数据库，方便查询历史对话记录。

# 技术方案

- 已经在项目中.env提供了，DOUBAO_API_KEY，DOUBAO_REALTIME_APP_ID，如果还需要其他参数请告知我。
- 想使用豆包新版的实时语音功能。
- 使用现有的技术架构。

# 功能验收标准
- 点击首页的主题对话，跳转到practice页面，点击话筒图标，可以支持实时语音对话
  - 实时对话，包括角色扮演，可以让用户选择不同的角色，如果能根据主题
  - 有暂停，重新语音的功能，还可以调节音量大小
  - 对话框能实时看见AI和客户的实时对话内容
- 对话结束后
  - 能导出pdf报告，包括中文相关的语法、词语、声调等问题。
- history页面，能查看历史的对话记录。
