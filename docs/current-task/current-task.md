# 作用域： apps/admin,apps/api

# 目标：开发对话场景管理模块，查询、新增、修改、删除

# 需求

- 查询功能
  - 查询条件：主题名，难度，类型，查询后分页加载，每页20条
- 新增，编辑对话主题
  - 对话主题图片，用文本框输入图片的链接
  - 可以新增编辑
- 删除
  - 对话主题可以删除，删除给二次确认，删除后刷新列表
- 初始化新对话场景数据
  - 类型：旅游 难度：低
  - 主题有：买飞机票，买衣服，交朋友，餐厅点餐，买水果，约朋友周末吃饭，买火车票

# 限制：不要看其他的docs，不用改动apps/web,

# 规范：

- 查询框，列表，分页，二次确认弹框，新增编辑表单都用公共组件。如果没有，则先封装到共同的组件

# 验证：

- 能根据查询条件查询对话主题的列表
- 查询、新增、修改、删除功能
- 初始化相关的数据
- 封装或使用共同的管理端的组件
