# 作用域： apps/admin，apps/api，apps/web

# 目标：web端增加google账号登录,注册,用户基础配置，登出功能，admin端增加用户管理功能

# 需求

- web端
  - 首页右上角增加，google账号登录注册，登录后可以点账号下拉按钮有登出，设置
  - 首页点击主题对话或者菜单，自动跳转登录或者注册
  - 登录需要鉴权功能，实时语音websocket接口需要登录鉴权
  - 用户登录后，可以右上角下拉选项的设置按钮，跳转配置页面配置基础信息，1.水平（低，中，高和对话主题水平一直）2.学习目标：值就是主题对话的类型 3.喜欢的豆包AI音色，下拉框选择
- 管理端
  - 增加用户管理模块，支持1.停用 2.查询 3.能修改一些信息
  - 管理端增加独立的登录页面，默认超级账号：admin/123456，可以查看管理端所有功能，并且有退出功能
  - 可以用户编辑页面配置基础信息，1.水平（低，中，高和对话主题水平一直）2.学习目标：值就是主题对话的类型 3.喜欢的豆包AI音色，下拉框选择
