# 作用域： apps/web

# 目标：优化首页样式，要简约的风格

# 需求

- header 左中右结构
  - 左边根据网站主题设计一个logo
  - 中部导航不变
  - 右边
    - 未登录，只要登录、注册
    - 已登录，只显示登录账号
- main主要区域
  - 整体风格要简约，减少页面元素及文字描述
  - 主要突出对话主题，
  - 减少文字描述，突出对话主题精美的图片
  - 对话主题的标签不显示角色名，显示难度
  - 点击主题卡片可以跳转到练习页，不需要卡片底部箭头的引导

# 限制：

- 要看其他的docs，不用改动apps/admin,尽量不要影响其他功能

# 规范：

- 仍然使用当前的packages/design-tokens的ui规范

# 验证：

- 页面风格简约，突出精美的对话主题图片作用，
- 点击对话主题可以跳转到练习页
- 符合当前的ui规范
